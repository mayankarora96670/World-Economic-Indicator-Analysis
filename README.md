World Economic Indicator Analysis
This project analyzes the World Economic Indicator Dataset to identify the key factors that influence GDP per Capita and sustainable economic growth. Using Python and data analysis techniques, the project uncovers relationships between economic indicators and provides insights that can help policymakers make informed decisions.

Problem Statement
The Organisation for Economic Co-operation and Development (OECD) aims to understand which factors contribute to sustainable economic growth. This project analyzes global economic indicators to determine the variables that have the greatest impact on GDP per Capita.

Dataset: World Economic Indicator Dataset

The dataset includes information such as:
GDP
GDP per Capita
Population
Inflation
Unemployment
Life Expectancy
Other economic indicators
